# Retail-Management-Database
SQL-based retail DB with DDL,DML,DQL,TCL,DCL, ER diagram


## 📌 Overview
This project is a SQL-based Retail Management System designed using PostgreSQL.  
It demonstrates database design, data manipulation, and access control using DCL.

---

## 🧩 Features
- ER Diagram Design
- Table Creation (DDL)
- Data Manipulation (DML)
- Data Querying (DQL)
- User Roles & Permissions (DCL)
- Transaction Control (TCL)

---

## 🏗️ Database Structure
Main tables included:
- Customers
- Products
- Orders
- Order_Details
- Inventory
- Suppliers

---

## 🔐 Roles & Access Control
- **Admin** → Full access to database  
- **Sales Staff** → Can view products and manage orders  
- **Inventory Manager** → Manages product stock and inventory  

---

- PostgreSQL
- pgAdmin

## 🚀 How to Run
1. Open PostgreSQL / pgAdmin  
2. Run `schema.sql` to create tables  
3. Run `dml_queries.sql` to insert data  
4. Run `dcl_queries.sql` to assign roles
